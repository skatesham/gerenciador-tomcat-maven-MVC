# gerenciador-tomcat-maven-MVC
WebApp Gerenciador com Tomcat e Maven seguindo Project Pattern MVC  
  
Esse Sistema possui:
1. Login
2. Cadastro de Empresas
3. Busca de Empresas
4. Logoff da conta

### Comando para execução  

```mvn tomcat7:run```
